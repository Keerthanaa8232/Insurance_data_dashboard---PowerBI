# INSURANCE DATA ANALYSIS - POWERBI DASHBOARD.

## Overview
This repository contains a Power BI project that provides interactive dashboards and visual analytics for data insights. The goal of this project is to transform raw data into meaningful visualizations that support informed business decisions.

 --------------------------------------------------------------------------------------------------------------------------------------------------------

## Features
- Dynamic dashboards with real-time filtering
- Advanced DAX calculations and measures
- Interactive KPIs, charts, and slicers
- Data modeling for performance optimization
- Integration with multiple data sources (Excel, SQL, APIs, etc.)

--------------------------------------------------------------------------------------------------------------------------------------------------------------
 
## Files in Repository
- **.pbix file:** The main Power BI project file
- **Data folder:** Contains sample datasets used in reports
- **Documentation folder:** Includes design notes, data dictionary, and screenshots
- **README.md:** Project documentation (this file)

--------------------------------------------------------------------------------------------------------------------------------------------------------------

## Insights and KPIs
Summarize key insights or metrics the dashboard highlights:
- Total revenue growth trends
